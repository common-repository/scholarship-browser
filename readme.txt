== Description ==

Scholarship Browser provides a German only Widget to display and search for high school student exchange scholarships. Students, schools and educational organizations might want to add this daily updated data source to help students to get abroad. 

The Widget will display in the sidebar with filtering options. Scholarships based on the filter can be selected and details can be viewed in the main content part.

In the widget admin menue there are further options:
1. Change the widget title
2. Select categories where the widget sould display
3. Apply this visibility to subcategories as well

The plugin requires jQuery to be available in Wordpress.

== Installation ==

1. Upload the folder `scholarship-browser` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the widget administration and drag "Scholarship Browser" to a sidebar, but only use one a time
4. Adjust the style.css to your needs

== Frequently Asked Questions ==

= Are the scholarships also available for British or American student? =

No, the scholarships are only for German, and some for Austrian and Swiss students.

= How do I have to update the dataset? =

The widget request always the most current data from the database at http://www.ausgetauscht.de

= How often is the database updated? =

There are ongoing updates made.
